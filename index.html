Guess My  Number !!!

A (single player) game filled with fun and curiosity.

Game made by  sravanTG

Welcome to the Game.

This game is designed using html, css and javascript. A web based Architecture.

The role of the player is to guess the number that is generated by the computer in the mystery box down below.

The mystery box can contain a number between (1 - 20).

The player Gets 5 chances. Scoring starts from 5 points. (decrement of -1 for every wrong guess)

Use the Hints below to find the right number.

Controls

RESET - resets the game to its original state including scores.

REPLAY - resets the game but preserves the Highscore.

VERIFY - checks your number with the number inside the mystery box.
